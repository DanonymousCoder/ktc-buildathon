Privacy Policy

Simple, local-first tracking.

FlowTrakka measures focused document activity without reading your document contents. Leaderboard sharing is optional and aggregate-only.

Effective date: July 22, 2026

What FlowTrakka Stores

FlowTrakka stores tracking settings, daily focus totals, document type metadata, last-read timestamps, and optional leaderboard preferences locally in your browser.

What FlowTrakka Does Not Store

FlowTrakka does not store document contents, keystrokes, unrelated browsing history, document titles in leaderboard payloads, document URLs in leaderboard payloads, or raw session history in leaderboard payloads.

Local Storage

By default, tracking data stays on your device using Chrome local extension storage. You can export your own data from the extension settings. Waitlist emails are not stored in browser local storage.

Waitlist Emails

If you submit an email through a FlowTrakka website form, the address is sent to the FlowTrakka backend and stored in a durable database for launch and product availability updates. Duplicate submissions update the existing subscriber record.

Leaderboard Sharing

Leaderboard sharing is disabled by default. If you opt in, FlowTrakka may send aggregate stats to the configured leaderboard backend, such as display name, user ID, focus totals, streak, document count, and document-type totals.

Chrome Permissions

FlowTrakka uses storage, tabs, idle, alarms, and configured leaderboard backend host access to detect supported document activity, avoid counting idle time, update focus totals, and sync opt-in aggregate rankings.

Remote Code

FlowTrakka does not execute remotely hosted code. Extension code is packaged with the installed extension.

Contact

For privacy questions, contact the FlowTrakka publisher through the contact email listed on the Chrome Web Store publisher profile.
